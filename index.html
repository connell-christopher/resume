About Me

Hi, I’m Connell Christopher — a bug bounty hunter and web security specialist. I focus on identifying real-world vulnerabilities using methodologies aligned with the OWASP Top 10 and practical training from platforms like PortSwigger Web Security Academy. I run PentraSec, a boutique web application security practice. Unlike large platforms, PentraSec is highly focused and hands-on — working closely with in-scope applications to uncover exploitable security issues before attackers do. My goal is simple: help developers and businesses stay secure by finding and reporting weaknesses early, so they can focus on building while I handle the security layer — from hidden attack paths to overlooked exposures that could impact trust and clients.
